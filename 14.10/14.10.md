# 14.10 その他のメソッド

- `s.strip` : `s` の先頭と末尾にある空白文字を削除する

    ```
    >> " Thank you. ".strip
    => "Thank you."
    ```

- `s.upcase` : `s` の小文字を大文字に変換した文字列を返す
- `s.upcase!` : `s` の小文字を大文字に変換する

    ```
    >> " Thank you. ".strip
    => "Thank you."
    >> str.upcase
    => "OBJECT-ORIENTED LANGUAGE"
    >> str
    => "Object-Oriented Language"
    >> 
    >> str.upcase!
    => "OBJECT-ORIENTED LANGUAGE"
    >> str
    => "OBJECT-ORIENTED LANGUAGE"
    ```
