# 7.4 メソッドの呼び出しの補足

```
>> def foo(a, b, c)
>> a + b + c
>> end
=> :foo
>> 
>> p foo(1, 2, 3)
6
=> 6
>> 
>> args1 = [2, 3]
=> [2, 3]
>> p foo(1, *args1)
6
=> 6
>> 
>> args2 = [1, 2, 3]
=> [1, 2, 3]
>> p foo(*args2)
6
=> 6
```

```
>> def foo(arg)
>> arg
>> end
=> :foo
>> 
>> p foo({"a" => 1, "b" => 2})
{"a"=>1, "b"=>2}
=> {"a"=>1, "b"=>2}
>> 
>> p foo("a" => 1, "b" => 2)
{"a"=>1, "b"=>2}
=> {"a"=>1, "b"=>2}
>> 
>> p foo(a: 1, b: 2)
{:a=>1, :b=>2}
=> {:a=>1, :b=>2}
```
