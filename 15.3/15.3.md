# 15.3 値を取り出す・設定する

```
>> h = Hash.new
=> {}
>> h["R"] = "Ruby"
=> "Ruby"
>> h["R"]
=> "Ruby"
```

`[]` を使って値を取り出す場合、登録されていないキーを指定するとデフォルト値が返される
デフォルト値以外を返すには `Hash#fetch` を使う

```
>> h = {"a" => "b", "c" => "d"}
=> {"a"=>"b", "c"=>"d"}
>> h.keys
=> ["a", "c"]
>> h.values
=> ["b", "d"]
>> h.to_a
=> [["a", "b"], ["c", "d"]]
>>
>> h.each_key {|key| p key}
"a"
"c"
=> {"a"=>"b", "c"=>"d"}
>> h.each_value {|value| p value}
"b"
"d"
=> {"a"=>"b", "c"=>"d"}
>> h.each {|key, value| p key, value}
"a"
"b"
"c"
"d"
=> {"a"=>"b", "c"=>"d"}
>> h.each {|item| p item}
["a", "b"]
["c", "d"]
=> {"a"=>"b", "c"=>"d"}
```
