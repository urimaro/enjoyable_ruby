# 15.4 あるオブジェクトをキーや値として持つか調べる

- `h.key?(key) : `key` をキーとして持っていれば `true` を返す
- `h.has_key?(key)` : `h.key?(key)` と同じ
- `h.include?(key)` : `h.key?(key)` と同じ
- `h.member?(key)` : `h.key?(key)` と同じ

```
>> h = {"a" => "b", "c" => "d"}
=> {"a"=>"b", "c"=>"d"}
>> h.key?("a")
=> true
>> h.has_key?("a")
=> true
>> h.include?("z")
=> false
>> h.member?("z")
=> false
```
