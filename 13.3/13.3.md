# 13.3 インデックスの使い方

```
>> alpha = ["a", "b", "c", "d", "e"]
=> ["a", "b", "c", "d", "e"]
>> alpha[1]
=> "b"
>> alpha[-1]
=> "e"
>> alpha[-2]
=> "d"
```

```
>> alpha[1..3]
=> ["b", "c", "d"]
>> alpha[1...3]
=> ["b", "c"]
>> alpha[1..7]
=> ["b", "c", "d", "e"]
>> alpha[1..]
=> ["b", "c", "d", "e"]
```

```
>> alpha[2, 2]
=> ["c", "d"]
>> alpha[2, 3]
=> ["c", "d", "e"]
```

```
>> alpha.at(1)
=> "b"
>> alpha.slice(1)
=> "b"
>> alpha.slice(1..3)
=> ["b", "c", "d"]
>> alpha.slice(2, 2)
=> ["c", "d"]
```

`a.at(n)` : `a[n]` と同じ
`a.slice(n)` : `a[n]` と同じ
`a.slice(n..m)` : `a[n..m]` と同じ
`a.slice(n, len)` : `a[n, len]` と同じ

```
>> alpha = ["a", "b", "c", "d", "e", "f"]
=> ["a", "b", "c", "d", "e", "f"]
>> alpha[1] = "B"
=> "B"
>> alpha[4] = "E"
=> "E"
>> alpha
=> ["a", "B", "c", "d", "E", "f"]
```
