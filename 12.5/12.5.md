# 12.5 数値型の変換

```
>> p 10.to_f
10.0
=> 10.0
>> p 10.8.to_i
10
=> 10
>> p -10.8.to_i
-10
=> -10
>> p "123".to_i
123
=> 123
>> p "12.3".to_f
12.3
=> 12.3
>> p "12.3".to_i
12
=> 12
```

`x.round(y)` : xの小数点以下y桁目を四捨五入した値を返す

```
>> 0.12.round(1)
=> 0.1
>> 0.18.round(1)
=> 0.2
>> 1.2.round
=> 1
>> 1.8.round
=> 2
>> 120.round(-2)
=> 100
>> 180.round(-2)
=> 200
```

`x.ceil(y)` : xより大きい最小の整数を返す
`x.floor(y)` : xより小さい最大の整数を返す

```
>> p 1.5.ceil
2
=> 2
>> p -1.5.ceil
-1
=> -1
>> p 1.5.floor
1
=> 1
>> p -1.5.floor
-2
=> -2
```
