# 16.6 正規表現を使うメソッド

- `string.sub(pattern, replace)` : `string` 中の、patternに「最初にマッチした部分」を `replace` に置き換えた文字列を返す
- `string.gsub(pattern, replace)` : `string` 中の、patternにマッチした全ての部分を `replace` に置き換えた文字列を返す

```
>> str = "abc   def  g    hi"
=> "abc   def  g    hi"
>> str.sub(/\s+/, " ")
=> "abc def  g    hi"
>> str.gsub(/\s+/, " ")
=> "abc def g hi"
```

ブロックを受け取ることもできる

```
>> str = "abracatabra"
=> "abracatabra"
>> str.sub(/.a/) do |matched|
?> "<" + matched.upcase + ">"
>> end
=> "ab<RA>catabra"
>> 
>> str.gsub(/.a/) do |matched|
?> "<" + matched.upcase + ">"
>> end
=> "ab<RA><CA><TA>b<RA>"
```
