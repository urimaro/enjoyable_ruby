# 15.6 キーと値を削除する

- `h.delete(key)` : キーが `key` の要素を削除し、削除した要素の値を返す  
    ブロックを指定すると、キーが存在しなかった場合、ブロックの実行結果を返す

    ```
    >> h = {"R" => "Ruby"}
    => {"R"=>"Ruby"}
    >> h["R"]
    => "Ruby"
    >> h.delete("R")
    => "Ruby"
    >> h["R"]
    => nil
    >> 
    >> h = {"R" => "Ruby"}
    => {"R"=>"Ruby"}
    >> h.delete("P") {|key| "no #{key}."}
    => "no P
    ```
