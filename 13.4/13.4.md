# 13.4 集合としての配列

```
>> ary1 = ["a", "b", "c"]
=> ["a", "b", "c"]
>> ary2 = ["b", "c", "d"]
=> ["b", "c", "d"]
>>
>> (ary1 & ary2)
=> ["b", "c"]
>> (ary1 | ary2)
=> ["a", "b", "c", "d"]
>> ary1 - ary2
=> ["a"]
```

```
>> num = [1, 2, 3]
=> [1, 2, 3]
>> even = [2, 4, 6]
=> [2, 4, 6]
>> 
>> num + even
=> [1, 2, 3, 2, 4, 6]
>> num | even
=> [1, 2, 3, 4, 6]
```
