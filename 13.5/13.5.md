# 13.5 列としての配列

キューやスタックにまつわる話

```
>> alpha = ["a", "b", "c", "d", "e"]
=> ["a", "b", "c", "d", "e"]
>> alpha.push("f")
=> ["a", "b", "c", "d", "e", "f"]
>> alpha.shift
=> "a"
>> alpha
=> ["b", "c", "d", "e", "f"]
```

```
>> alpha = ["a", "b", "c", "d", "e"]
=> ["a", "b", "c", "d", "e"]
>> alpha.push("f")
=> ["a", "b", "c", "d", "e", "f"]
>> alpha.pop
=> "f"
>> alpha
=> ["a", "b", "c", "d", "e"]
```
