# 14.1 文字列を作る

```
>> str1 = "これも文字列"
=> "これも文字列"
>> str2 = 'あれも文字列'
=> "あれも文字列"
>> 
>> moji = "文字列"
=> "文字列"
>> str1 = "あれも#{moji}"
=> "あれも文字列"
>> str2 = 'あれも#{moji}'
=> "あれも\#{moji}"
```

```
>> %Q(Rubyの文字列には「''」も「""」も使われます。)
=> "Rubyの文字列には「''」も「\"\"」も使われます。"
>> %q|Ruby said, 'Hello, world!'|
=> "Ruby said, 'Hello, world!'"
```

`%Q` : 「""」で囲った文字列
`%q` : 「''」で囲った文字列
