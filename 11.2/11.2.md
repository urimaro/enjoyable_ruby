# 11.2 ブロックの使われ方

```
>> alphabet = ["a", "b", "c", "d", "e"]
=> ["a", "b", "c", "d", "e"]
>> alphabet.each do |i|
?> puts i.upcase
>> end
A
B
C
D
E
=> ["a", "b", "c", "d", "e"]
```
