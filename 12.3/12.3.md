# 12.3 算術演算

```
>> p 1 + 1
2
=> 2
>> p 1 + 1.0
2.0
=> 2.0
>> p 2 - 1
1
=> 1
>> p 2 - 1.0
1.0
=> 1.0
>> p 3 * 2
6
=> 6
>> p 3 * 2.0
6.0
=> 6.0
>> p 3 * -2.0
-6.0
=> -6.0
>> p 5 / 2
2
=> 2
>> p 5 / 2.0
2.5
=> 2.5
>> p 5 % 2
1
=> 1
>> p 5 % 2.0
1.0
=> 1.0
>> p 5 ** 2
25
=> 25
>> p 5 ** 0.5
2.23606797749979
=> 2.23606797749979
>> p 8 ** (1 / 3r)
2.0
=> 2.0
```

```
>> r = (2 / 5r) + (1 / 3r)
=> (11/15)
>> p r
(11/15)
=> (11/15)
>> p r.to_f
0.7333333333333333
=> 0.7333333333333333

>> c = (1 + 2i) ** 2
=> (-3+4i)
>> p c
(-3+4i)
=> (-3+4i)

```
>> p 5 ** -2.0
0.04
=> 0.04

(1/25)
