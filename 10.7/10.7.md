# 10.7 例外処理の構文の補足

P213 L1

> 例外処理のためにメソッドの処理全体をbegin〜endでくくるような場合は、beginとendを省略してrescue節やensure節を記述することができます。

```
def foo
  メソッドの本体
rescue => ex
  例外処理
ensure
  後処理
end
```

P213 L10

> 同様にクラス定義内でも、rescue節やensure節を記述できます。
> ただし、クラス定義内で例外が発生すると、例外が発生した箇所以降のメソッド定義などが行われなくなるため、通常はこの処理を利用することはありません。

```
class Foo
  クラス定義の本体
rescue => ex
  例外処理
ensure
  後処理
end
```
