ローカル変数はブロックの中でも扱える
ブロック内で初めて出てきた変数はブロック外で扱えない

```
goh@goh ~/s/g/u/enjoyable_ruby> ruby 11.4/local_and_block2.rb
[1, 1, 1]
[2, 2, 2]
[3, 3, 3]

[0, 0, 3]
```

> ブロックローカル変数は、ブロック変数の後ろに「;」で区切って定義します。
