# 11.4 ローカル変数とブロック変数

```
goh@goh ~/s/g/u/enjoyable_ruby> ruby 11.4/local_and_block.rb

```

```
goh@goh ~/s/g/u/enjoyable_ruby> ruby 11.4/local_and_block_error.rb
```

ローカル変数はブロックの中でも扱える
ブロック内で初めて出てきた変数はブロック外で扱えない
