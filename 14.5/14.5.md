# 14.5 文字列を比較する

```
>> "aaa" == "baa"
=> false
>> "aaa" == "aa"
=> false
>> "aaa" == "aaa"
=> true
>> "aaa" != "baa"
=> true
>> "aaa" != "aaa"
=> false
```

```
>> "aaaaa" < "b"
=> true
>> "かけ" < "かこ"
=> true
>> "かけ" < "がけ"
=> true
>> "かこ" < "がけ"
=> true
>> str = %w(かけ がけ かこ)
=> ["かけ", "がけ", "かこ"]
>> str.sort
=> ["かけ", "かこ", "がけ"]
```

文字列の大小は、文字コード順で決まる
文字コード順は50音順ではない
