# 15.8 2つのハッシュを合わせる

- `h.merge(hash)` : `h` の末尾に `hash` を追加したハッシュを返す
- `h.merge!(hash)` : `h` の末尾に `hash` を追加し、自身を返す
- `h.update(hash)` : `h.merge!(hash)` と同じ

```
>> h = {"a" => "x"}
=> {"a"=>"x"}
>> h.merge({"b" => "y"})
=> {"a"=>"x", "b"=>"y"}
>> h
=> {"a"=>"x"}
>> 
>> h.merge!({"b" => "y"})
=> {"a"=>"x", "b"=>"y"}
>> h
=> {"a"=>"x", "b"=>"y"}
>> 
>> h = {"a" => "x"}
=> {"a"=>"x"}
>> h.update({"b" => "y"})
=> {"a"=>"x", "b"=>"y"}
>> h
=> {"a"=>"x", "b"=>"y"}
```
