# 20.4 時刻のフォーマット

- `time.strftime(format)` : `format` で指定した書式に従い、`time` を文字列にして返す
    - `format` : フォーマット  
        フォーマットで使える文字列はP431の一覧参照
- `time.to_s` : 以下と同じ  
    `time.strftime("%Y-%m-%d %H:%M:%S %z")`

```
>> t = Time.new
=> 2020-01-11 14:32:26 +0900
>> t.to_s
=> "2020-01-11 14:32:26 +0900"
>> t.strftime("%Y-%m-%d %H:%M:%S %z")
=> "2020-01-11 14:32:26 +0900"
```
