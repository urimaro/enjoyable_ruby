# 14.9 文字列と配列で共通するメソッド

```
>> str = "abcde"
=> "abcde"
>> str[2, 1] = "C"
=> "C"
>> str
=> "abCde"
>> 
>> str = "abcde"
=> "abcde"
>> str.slice(2, 3)
=> "cde"
>> str
=> "abcde"
>> 
>> "こんにちはRuby".byteslice(15, 4)
=> "Ruby"
```

```
>> str = "Hello, Ruby."
=> "Hello, Ruby."
>> str.slice!(-1)
=> "."
>> str
=> "Hello, Ruby"
>> str.slice!(5..6)
=> ", "
>> str.slice!(0, 5)
=> "Hello"
>> str
=> "Ruby"
```
