# 15.7 ハッシュを初期化する

- `h.clear` : 要素を空にする

    ```
    >> h = {"a" => "b", "c" => "d"}
    => {"a"=>"b", "c"=>"d"}
    >> h.clear
    => {}
    >> h
    => {}
    >> 
    >> h = {"k1" => "v1"}
    => {"k1"=>"v1"}
    >> g = h
    => {"k1"=>"v1"}
    >> h.clear
    => {}
    >> h
    => {}
    >> g
    => {}
    >> 
    >> h = {"k1" => "v1"}
    => {"k1"=>"v1"}
    >> g = h
    => {"k1"=>"v1"}
    >> h = Hash.new
    => {}
    >> h
    => {}
    >> g
    => {"k1"=>"v1"}
    ```

**メソッドは、変数に対してではなく、変数が参照しているオブジェクトに対して作用する**

```
>> h = {"A" => {"a" => "x", "b" => "y"},
?> "B" => {"a" => "v", "b" => "w"}}
=> {"A"=>{"a"=>"x", "b"=>"y"}, "B"=>{"a"=>"v", "b"=>"w"}}
>> h["A"]["a"]
=> "x"
>> h["B"]["a"]
=> "v"
```
