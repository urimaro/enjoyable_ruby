# 20.7 日付を取得する

```
>> require "date"
=> true
>> 
>> d = Date.today
=> #<Date: 2020-01-12 ((2458861j,0s,0n),+0s,2299161j)>
>> 
>> d.year
=> 2020
>> d.month
=> 1
>> d.day
=> 12
>> d.wday
=> 0
>> d.mday
=> 12
>> d.yday
=> 12
>> 
>> d = Date.new(2018, 11, 19)
=> #<Date: 2018-11-19 ((2458442j,0s,0n),+0s,2299161j)>
>> 
>> d = Date.new(2020, 2, -1)
=> #<Date: 2020-02-29 ((2458909j,0s,0n),+0s,2299161j)>
```

日に `-1` を指定すると、月末の日付になる
