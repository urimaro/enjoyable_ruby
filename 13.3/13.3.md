# 13.3 インデックスの使い方

```
>> alpha = ["a", "b", "c", "d", "e"]
=> ["a", "b", "c", "d", "e"]
>> alpha[1]
=> "b"
>> alpha[-1]
=> "e"
>> alpha[-2]
=> "d"
```

```
>> alpha[1..3]
=> ["b", "c", "d"]
>> alpha[1...3]
=> ["b", "c"]
>> alpha[1..7]
=> ["b", "c", "d", "e"]
>> alpha[1..]
=> ["b", "c", "d", "e"]
```

```
>> alpha[2, 2]
=> ["c", "d"]
>> alpha[2, 3]
=> ["c", "d", "e"]
```
