# 19.3 Encodingクラス

- `string.encode(encoding)` : `string` のエンコーディングを `encoding` に変換した文字列を返す
    - `encoding` : エンコーディング名の文字列

    ```
    >> str = "こんにちは"
    => "こんにちは"
    >> str.encoding
    => #<Encoding:UTF-8>
    >> str2 = str.encode("Shift_JIS")
    => "\x{82B1}\x{82F1}\x{82C9}\x{82BF}\x{82CD}"
    >> str2.encoding
    => #<Encoding:Shift_JIS>
    ```
