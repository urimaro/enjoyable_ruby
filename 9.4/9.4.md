# 9.4 範囲演算子

```
>> p (5..10).to_a
[5, 6, 7, 8, 9, 10]
=> [5, 6, 7, 8, 9, 10]
>> p (5...10).to_a
[5, 6, 7, 8, 9]
=> [5, 6, 7, 8, 9]
>> 
>> p ("a".."f").to_a
["a", "b", "c", "d", "e", "f"]
=> ["a", "b", "c", "d", "e", "f"]
>> p ("a"..."f").to_a
["a", "b", "c", "d", "e"]
=> ["a", "b", "c", "d", "e"]
>> 
>> ("a".."f").class
=> Range
>> ("a"..."f").class
=> Range
>> 
>> p ("a".."f")
"a".."f"
=> "a".."f"
>> p ("a"..."f")
"a"..."f"
=> "a"..."f"
```
