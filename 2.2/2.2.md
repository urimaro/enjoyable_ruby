# 2.2 ハッシュ(Hash)

```
goh@goh ~/s/g/u/enjoyable_ruby> irb --simple-prompt
>> sym = :foo
=> :foo
>> sym.to_s
=> "foo"
>> "foo".to_sym
=> :foo
```

```
goh@goh ~/s/g/u/enjoyable_ruby> irb --simple-prompt
>> address = {name: "高橋", furigana: "タカハシ"}
=> {:name=>"高橋", :furigana=>"タカハシ"}
>> address[:name]
=> "高橋"
>> address[:furigana]
=> "タカハシ"
>> address[:tel] = "000-1234-5678"
=> "000-1234-5678"
>> address
=> {:name=>"高橋", :furigana=>"タカハシ", :tel=>"000-1234-5678"}
```
