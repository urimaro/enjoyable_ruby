# 16.5 キャプチャ

キャプチャ : 正規表現にマッチした部分の一部を取り出すこと
    `$数字` の形式で参照する

```
>> /(.)(.)(.)/ =~ "abc"
=> 0
>> first = $1
=> "a"
>> second = $2
=> "b"
>> third = $3
=> "c"
```

- `(?: )` : キャプチャする必要のないパターンを指定する

    ```
    >> /(.)(\d\d)+(.)/ =~ "123456"
    => 0
    >> $1
    => "1"
    >> $2
    => "45"
    >> $3
    => "6"
    >> 
    >> /(.)(?:\d\d)+(.)/ =~ "123456"
    => 0
    >> $1
    => "1"
    >> $2
    => "6"
    ```
